I wrote the programme in PyCharm IDE and ran in Windows Machine

Running the program: 
i) Open the folder in PyCharm. Have`server.py` file as an active file.
ii) Under `Run` dropdown option, click on edit configuration.
iii) In `Script parameters` field, write localhost and port number. 
eg: 127.0.0.1 4000
iv) Press 'OK'
v) Run the server.py file.
vi) Open browser, and in address bar write file you want to access. 
eg. 127.0.0.1:4000/f1.html
vii) Fill the form and press submit button.
viii) It should redirect you to respone form with appropriate value.
