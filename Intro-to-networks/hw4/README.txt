a) File1: server-w\o-select.py
This file can handle only one connection.
In console the following command is used to run the file:
python3 server-w\o-select.py 127.0.0.1 3000


b) File2 : server.py
This file can handle connections upto 5.
In console the following command is used to the file:
python3 server.py 127.0.0.1 3000


General format of commandline to run in console is:
python3 filename localhost portnumber
